##  项目简介 (Introduction)

**MineSweeper_win7_CN** 是一个经典 Windows 7 风格扫雷游戏的复刻版本，专为中文用户优化。本项目旨在重现 Win7 时代标志性的 Aero 玻璃界面风格与流畅的游戏体验，让大家在现代操作系统上也能重温经典的扫雷乐趣。

##  功能特性 (Features)

* **经典还原**：完美复刻 Windows 7 扫雷的 UI 界面、图标与音效。
* **全中文界面**：菜单、设置及游戏提示均已汉化，不仅是简单的翻译，更符合中文用户习惯。
* **多难度选择**：
    * **初级**：9x9 网格，10 个雷
    * **中级**：16x16 网格，40 个雷
    * **高级**：30x16 网格，99 个雷
    * **自定义**：支持自定义网格大小与地雷数量
* **智能操作**：支持鼠标左右键双击（Chording）快速开启周围方块。
* **统计功能**：(如果是功能之一) 记录最佳时间和胜率。
